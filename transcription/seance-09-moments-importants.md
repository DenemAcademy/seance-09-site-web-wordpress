## Prompts et commandes identifiés
### Installer Claude Code
```text
npm install -g @anthropic-ai/claude-code
```
### Lancer Claude Code dans le dossier site
```text
claude --dangerously-skip-permissions
```
### Ouvrir / vérifier les serveurs MCP
```text
/mcp
```
### Connecter Claude à WordPress sans donner le mot de passe
```text
Ouvre un nouvel onglet Chrome via le MCP et navigue vers https://[TON_DOMAINE]/wp-admin. Attends que je me connecte à la main, puis dis-moi quand tu vois le dashboard.
```
### Créer la page de travail
```text
Crée une nouvelle page WordPress appelée "Accueil" et ouvre-la dans l’éditeur Elementor.
```
### Prompt de correction court
```text
Le bouton n’est pas parfait, améliore-le stp. Vérifie desktop, tablette et mobile avant de sauvegarder.
```
### Prompt section suivante
```text
[Image #X] Voici la prochaine section. Reprends le même process : extraction source, plan, construction Elementor, vérification responsive, puis sauvegarde.
```
### Prompt contexte SEO
```text
Analyse mon audit SEO et ma stratégie. Réécris les textes de cette section pour RestIA, en gardant la même structure visuelle et un ton simple.
```
