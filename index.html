Prompt à copier
PROMPT TECHNIQUE - RECONSTRUCTION PIXEL-PERFECT D'UNE SECTION D'UN SITE SOURCE VERS ELEMENTOR

CONTEXTE
3 onglets Chrome :
- SOURCE : [URL_SOURCE] - section ciblée : [NOM_SECTION]
- EDITEUR : [URL_ELEMENTOR]
- PREVIEW : [URL_PREVIEW]

OBJECTIF
Reproduire la section [NOM_SECTION] dans Elementor avec une fidélité visuelle pixel-perfect sur desktop 1440, tablette 768 et mobile 390.
Tu dois reprendre la structure, la typographie, les espacements, les couleurs, les images, les hover, les animations et le comportement responsive.

CONTRAINTES
- Widgets natifs Elementor uniquement : Container, Heading, Text Editor, Button, Image, Icon, Icon Box, Divider, Tabs, Accordion, Form si nécessaire.
- Pas de widget HTML, pas de shortcode, pas de plugin tiers sans validation.
- CSS custom seulement si un effet est impossible avec Elementor natif. Tu me préviens avant.
- Toutes les images visibles dans la source doivent être présentes dans la section finale.
- Tu adaptes les textes à mon projet si je fournis un texte ou un audit SEO.

ETAPE 1 - EXTRACTION SOURCE
Localise le conteneur racine de la section.
Extrait pour chaque élément : texte, classe, position, dimensions, style calculé, couleurs, polices, padding, margin, border, radius, shadow, transform, animations et pseudo-éléments.
Refais l'analyse en 1440, 768 et 390.
Liste toutes les images avec URL, taille et alt.

ETAPE 2 - DETECTION DES EFFETS
Teste les hover, transitions, animations temporelles, changements au scroll et accordéons/tabs si présents.
Pour chaque effet, propose la solution Elementor native ou la meilleure approximation.

ETAPE 3 - PLAN D'IMPLEMENTATION
Avant de toucher Elementor, écris le plan : conteneur racine, enfants, widgets, responsive, assets et effets.
Si une décision est incertaine, pose une question courte.

ETAPE 4 - CONSTRUCTION ELEMENTOR
Construis avec les containers et widgets natifs.
Sauvegarde à chaque jalon important, pas à chaque micro-modification.

ETAPE 5 - VERIFICATION
Recharge la preview.
Compare source et résultat en desktop, tablette et mobile.
Liste les écarts visibles, corrige-les, puis recommence jusqu'à obtenir une version propre.

GO
Lance l'extraction de la section maintenant. Tu ne me rends la main qu'après vérification responsive ou blocage clair.
